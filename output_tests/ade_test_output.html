This figure is designed to visually explain the concept of an equation by showing how two expressions are set equal to each other.

A solution to an equation is a number that makes the equation true when substituted for the unknown. Another way to think of this is to ask the question, “For what value of $x$ does the expression $x + 5$ evaluate to 7?”
